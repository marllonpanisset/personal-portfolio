Aprender tecnologia exige consistência.

Atualmente estou focando meus estudos em diferentes áreas para construir uma base mais completa como profissional de TI.

## Desenvolvimento

Continuo estudando desenvolvimento de aplicações utilizando tecnologias modernas.

Entre os principais assuntos:

- React;
- Next.js;
- TypeScript;
- criação de APIs;
- boas práticas de desenvolvimento.

Meu objetivo é continuar evoluindo minha capacidade de construir soluções reais.

## Python

Também estou aprofundando meus conhecimentos em Python.

A linguagem possui uma grande importância em áreas como:

- automação;
- scripts;
- análise de dados;
- backend;
- segurança.

## Banco de dados

Outro ponto importante dos meus estudos é SQL.

Entender dados é essencial para qualquer profissional que trabalha com sistemas.

Estou estudando conceitos como:

- consultas;
- relacionamentos;
- modelagem;
- organização de informações.

## Fundamentos de TI

Além do desenvolvimento, estou buscando fortalecer conhecimentos fundamentais:

- sistemas operacionais;
- redes;
- infraestrutura;
- suporte técnico;
- troubleshooting.

Acredito que essa combinação entre desenvolvimento e fundamentos de TI cria uma visão mais completa sobre tecnologia.
